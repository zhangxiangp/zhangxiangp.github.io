## About Me

I am a researcher at the School of Computer Science and Engineering, Hunan University of Science and Technology. My research focuses on software engineering, particularly code intelligence (code search, code completion, code clone detection) and intelligent web service computing. I have published 25+ papers in journals and conferences including *Applied Intelligence*, *Pattern Recognition*, *Journal of Systems and Software*, *Journal of Parallel and Distributed Computing*, *Concurrency and Computation: Practice and Experience*, *Information and Software Technology*, and *Software Journal* (in Chinese), with 340+ citations and an h-index of 6.

## Research Interests

- **Code Intelligence:** code search, code completion, code clone detection, code representation learning
- **Web Service Computing:** service classification, service recommendation, API ecosystem analysis
- **Deep Learning for SE:** Transformer-based models, attention mechanisms, pointer networks for source code
- **Generative AI & Human Factors:** impact of GenAI on software development workflows
