# Xiangping Zhang Academic Website

This repository hosts the GitHub Pages source for <https://zhangxiangp.github.io>.

The site uses the [al-folio](https://github.com/alshedivat/al-folio) Jekyll theme. Publications are maintained in `_bibliography/papers.bib` and rendered automatically on `/publications/`.

## Local Preview

```bash
docker compose up
```

Then open <http://localhost:8080>.
