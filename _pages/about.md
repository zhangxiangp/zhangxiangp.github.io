I am Xiangping Zhang, a researcher in **software engineering** and **code intelligence**. My work focuses on applying deep learning techniques to software engineering tasks, including code search, code completion, code clone detection, and intelligent web service computing (classification, clustering, and recommendation).

I have published 25+ papers in journals and conferences such as *Applied Intelligence*, *Journal of Parallel and Distributed Computing*, *Pattern Recognition*, *Journal of Systems and Software*, *Concurrency and Computation: Practice and Experience*, *软件学报*, *计算机学报*, IEEE SCC, IEEE ICWS, and CollaborateCom. My research has received 340+ citations with an h-index of 6 (Google Scholar).

### Research Interests

- **Code Intelligence** — code search, code completion, code clone detection, code representation
- **Web Service Computing** — service classification, clustering, and recommendation
- **Deep Learning for SE** — Transformer, attention mechanisms, graph neural networks
- **Generative AI & Human Factors** — LLM-based agents, developer experience with GenAI tools

For a complete list of papers, see [publications](/publications/).
